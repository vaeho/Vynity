Prediction Process

Vynity offers two types of predictions for hydro powerplants:

  • Short-term Predictions: These predictions are based on recent historical data and provide forecasts for the next 48 hours.
  • Long-term Predictions: These predictions utilize a more extensive historical dataset and are designed to provide estimates for the next month.

Prediction Process Guide

  • Upload an Excel file containing historical hydro powerplant output data.
  • Select the sheet within the Excel file that contains the relevant data.
  • Choose the plant or column from the selected sheet for which you want to generate predictions.
  • View the predicted output for the selected plant in a chart and preview table.
  • Download the complete prediction results in an Excel file format.
